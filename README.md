# Lumara - Luz y Aroma que Abrazan 🕯️

Bienvenido al repositorio de **Lumara**, una plataforma web dedicada a la exhibición y personalización de velas artesanales elaboradas con cera de soja premium.

## ✨ Sobre el Proyecto
Este sitio web ha sido diseñado para ofrecer una experiencia visual elegante y minimalista, resaltando la esencia artesanal de nuestros productos. Permite a los usuarios explorar nuestras colecciones de temporada y personalizar cada detalle de su pedido.

### Características principales:
* **Catálogo Interactivo:** Visualización de colecciones (Decoración, Eventos, Regalos).
* **Diseño Responsive:** Optimizado para visualizarse correctamente en dispositivos móviles y computadoras.
* **Conexión Directa:** Integración con WhatsApp para pedidos personalizados.
* **Estética Minimalista:** Uso de tipografías elegantes (Montserrat y Playfair Display) y una paleta de colores sofisticada.

## 🚀 Tecnologías Utilizadas
* **HTML5:** Estructura semántica del sitio.
* **CSS3:** Estilos personalizados, incluyendo el uso de Flexbox y transiciones suaves.
* **JavaScript:** Interactividad y manejo de eventos.
* **Google Fonts:** Tipografías para una identidad visual única.

## 📂 Estructura del Repositorio
* `index.html`: Página principal de bienvenida y hero section.
* `decoracion.html`, `eventos.html`, `regalos.html`: Secciones específicas del catálogo.
* `css/`: Carpeta con los archivos de estilos (`style.css`, etc.).
* `js/`: Scripts para la funcionalidad del sitio.
* `imagenes/`: Recursos gráficos y fotografías de las velas.
